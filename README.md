# CareQuest Pilot Validation

This repository tracks early validation for CareQuest’s AI-native neurodivergence care simulation prototype.

## Validation Goal

Understand whether learners find behavioral simulations useful for improving certification readiness and practical care confidence.

## First Test

5–10 early users testing the AI Care Simulation prototype.

## What We Measure

- usability
- perceived usefulness
- simulation clarity
- feedback quality
- willingness to try more missions
- relevance to ABAT/RBT readiness

## Core Hypothesis

Learners will find simulation-based practice more useful than passive certification content.
