# Testing Script

## Instructions For Tester

1. Open the AI Care Simulation prototype.
2. Select one scenario.
3. Write how you would respond.
4. Click evaluate.
5. Read the feedback.
6. Share your honest thoughts.

## Questions

1. Was the scenario realistic?
2. Was the feedback useful?
3. Did you understand what a better response looks like?
4. Would this help someone preparing for ABAT/RBT?
5. Would you try more simulations?
6. What should be improved?
